# udemy-nestjs-course

My solution of the exercises proposed in the course NestJS: The Complete Developer's Guide by Stephen Grider and Udemy.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
